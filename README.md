# biy-daalt-11
B242270132 М. Хуншагай
Хийсэн ажлын тойм:
Энэхүү бие даалтын хүрээнд номын сангийн үйл ажиллагааг автоматжуулах цогц системийг хөгжүүлсэн. Б хэсэгт Express.js болон Node.js ашиглан ном, гишүүн, зээлэлтийн мэдээллийг удирдах REST API-г RFC 7807 стандартын дагуу боловсруулсан.
Систем нь ном зээлэхэд нэг хүн 5-аас дээш ном авах боломжгүй байх, хугацаа сунгах, хайлт болон pagination хийх зэрэг бизнес дүрмүүдийг бүрэн хангасан. Мөн бүх endpoint-уудыг Postman-аар автомат тест хийж, баримтжуулсан.
Ажиллуулах заавар:
Б хэсэг
1-partB хавтас руу орнр
2-npm install
3-npm start
Тулгарсан бэрхшээл ба шийдэл:
Асуудал: Алдаа гарахад ямар алдаа гарч байгааг хэрэглэгчид ойлгомжтой харуулах шаардлага гарсан.
Шийдэл: RFC 7807 гэдэг "стандарт" ашиглаж алдаа болгоныг тодорхой тайлбарладаг болгосон.
Дүгнэлт:
Энэ ажлыг хийх явцад вэб болон API хоёр хоорондоо хэрхэн харилцдаг, токен ашиглаж хэрхэн "нууц үг" шалгадаг талаар маш сайн ойлгож авлаа. Цаашид энэ системээ илүү гоё вэб дизайнтай болгож, жинхэнэ өгөгдлийн сантай холбож сайжруулбал зүгээр санагдсан.
